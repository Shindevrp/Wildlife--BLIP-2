# Learn & Apply: Wildlife BLIP-2 Project

This guide provides resources and learning paths to help you understand, use, and extend the Wildlife BLIP-2 image captioning and VQA project.

---

## 1. Core Concepts

- **BLIP-2 Model**: [BLIP-2 Paper](https://arxiv.org/abs/2301.12597) | [Hugging Face Model Card](https://huggingface.co/Salesforce/blip2-flan-t5-xl)
- **Image Captioning**: [Image Captioning Overview (Stanford CS231n)](https://cs231n.github.io/captioning/)
- **Visual Question Answering (VQA)**: [VQA Challenge](https://visualqa.org/)

---

## 2. Tutorials & Guides

- **Hugging Face Transformers**: [Official Docs](https://huggingface.co/docs/transformers/index)
- **Fine-tuning Vision-Language Models**: [Hugging Face Vision-Language Course](https://huggingface.co/learn/nlp-course/chapter7/7?fw=pt)
- **Streamlit Web Apps**: [Streamlit Docs](https://docs.streamlit.io/)
- **PyTorch Basics**: [PyTorch Tutorials](https://pytorch.org/tutorials/)

---

## 3. Applying to Your Data

- Prepare your dataset in the format shown in `README.md`.
- Use `finetune_blip2.py` to train on your images and captions/questions.
- Use `app.py` to interact with your model via a web interface.
- For deployment, see [ONNX](https://onnx.ai/) and [TorchScript](https://pytorch.org/docs/stable/jit.html) guides.
